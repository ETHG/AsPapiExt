
%areashop_num_regions_for_rent% : Number of rent regions avaliable to be rented by a tenant. 
%areashop_num_regions_for_sale% : Number of rent regions avaliable to be purchased by a buyer
%areashop_num_regions_avaliable% : Number of buy and rent regions avaliable to be bought or rented.

%areashop_num_regions% : Total number of regions

%areashop_num_regions_bought% : Number of buy regions that currently have an owner
%areashop_num_regions_rented% : Number of rental regions that currently have a tenant
%areashop_num_regions_occupied : Number of buy and rent regions with an owner or tenant


%areashop_list_regions_for_rent% : List of rent regions avaliable to be rented
%areashop_list_regions_for_sale% : List of buy regions avaliable to be purchased
%areashop_list_regions_avaliable% : List of buy or rent regions avaliable to be purchased or rented

%areashop_list_regions% : List of all regions that can be bought or rented

%areashop_list_regions_bought : List of buy regions avaliable to be purchased
%areashop_list_regions_rented% : List of rental regions avaliable to be rented
%areashop_list_regions_occupied : List of regions currently bought or rented


%areashop_num_regions_bought_own% : Number of buy regions the player owns
%areashop_num_regions_rented_own% : Number of rental regions the player rents
%areashop_num_regions_occupied_own% : Number of regions the player buys or rents

%areashop_list_regions_bought_own% : List of the buy regions the player owns 
%areashop_list_regions_rented_own% : List of rental regions the player rents
%areashop_list_regions_occupied_own% : List of regions the player buys or rents

%areashop_num_price_owed% : Total amount player pays for all the regions they own
%areashop_num_price_payed% : Total amount the payer has played for the regions they own

%areashop_list_price_owed% : List of rent costs for regions rented by the player
%areashop_list_price_payed% : List of prices payed for regions bought by the player
%areashop_list_price% : List of rent and purchase prices of regions rented and purchased by the player
